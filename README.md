# Cosmatic Store (Web project)

This is a React.js project built with **Tailwind CSS** and **React Router**.  
The project includes multiple pages, a responsive navbar, and basic routing.

## Features

- **React Router** for client-side routing
  - Home page
  - Products page
  - Contact page
  - Login page
  - 404 Not Found page
- **Tailwind CSS** for styling
- Responsive design for mobile and desktop
- Reusable components:
  - Navbar
  - Product items
  - Contact card
- Login page with navigation to Home page (demo, no authentication)
